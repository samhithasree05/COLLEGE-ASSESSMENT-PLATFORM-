Online Assessment Platform for College

An Online Assessment Platform built using HTML, CSS, and JavaScript for conducting and evaluating tests in a simple and user-friendly way. Designed as a lightweight solution for college-level hackathons and projects.

Features:
Instructor Section
Create exams with multiple-choice questions.
Save exams locally (in localStorage).
View all created exams.
Delete exams when no longer needed.

Student Section:
View and take available exams.
Save answers for individual questions without submitting the full test.
Submit completed exams to get instant results.
Friendly toast-style notifications instead of browser alerts.

UI/UX:
Clean, responsive design using plain CSS.
Modern notification system replaces intrusive alert popups.
Works entirely offline (no server required).
